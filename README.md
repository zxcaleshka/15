# Игра "15"

## Описание проекта
Игра "15" — это головоломка на квадратном поле размером 4x4, состоящая из плиток с числами от 1 до 15 и одной пустой клеткой.
Цель игры — упорядочить плитки по возрастанию, перемещая их на свободное место.
